# Images Directory

## Adding Your Profile Photo

1. Add your profile photo to this directory with the name `profile-photo.jpg`
2. Recommended specifications:
   - Size: 400x400 pixels or larger
   - Format: JPG, PNG, or WebP
   - Square aspect ratio works best
   - High quality/resolution for crisp display

## Current Images
- `profile-photo.jpg` - Your profile photo (to be added)

The photo will automatically display in the hero section once added.

`profile-photo.jpg` is also referenced as **`og:image`** / Twitter card image in `client/index.html`. If you use a custom domain or Cloudflare Pages URL, update the absolute image URLs in `index.html`, `robots.txt`, and `sitemap.xml` to match your live site.
